## How to Upload:
1. Go to https://github.com/jonkin478/Jachin
2. Click "Add file" > "Upload files"
3. Select images to upload
4. Commit changes
5. GitHub will provide URLs in the format:
   `https://raw.githubusercontent.com/jonkin478/Jachin/main/images/filename.jpg`
